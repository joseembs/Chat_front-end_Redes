Esta é uma versão do aplicativo cuja única diferença é que
o IP com o qual o cliente se comunica é o "127.0.0.1", já
que assim é possível testá-lo localmente.

Única configuração que precisa ser feita pelo usuário é 
mudar localmente o IP na linha 72 do "server.py" de 
"26.212.49.229" para "127.0.0.1" ou "0.0.0.0" antes de 
iniciá-lo, para que ele receba corretamente os dados do 
cliente.

Arquivos no servidor serão salvos na mesma pasta do 
"server.py" e arquivos do cliente ficarão na mesma pasta 
do executável.
